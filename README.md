# UHC Essentials: Forge 1.8.9

A port of [UHC Essentials](https://github.com/abandonedaccount6235/UHC-Essentials) by sintinium to Minecraft 1.8.9 via Forge.

> **Status:** Work in progress.

## What it does

Adds a customizable HUD with overlays:

- **Coordinates:** X/Y/Z position
- **Biome:** current biome name
- **FPS:** current frame rate
- **Arrow counter:** inventory arrow count
- **Compass:** compass item pointing toward (0,0)
- **Clock:** clock item with time-of-day indicator
- **Armor status:** current armor durability

All windows are draggable, individually toggleable, and fully color-customizable with per-window RGBA settings. Configuration is saved between sessions.

## Requirements

- Minecraft 1.8.9
- Forge 11.15.1.1722
- Java 8
- OptiFine compatible

## Original mod

This is a Forge port of the [original UHC Essentials mod by sintinium](https://www.reddit.com/r/ultrahardcore/comments/3mel2w/uhc_essentials_new_download_link/). The original source code and compiled files are preserved in `reference/` for reference.
